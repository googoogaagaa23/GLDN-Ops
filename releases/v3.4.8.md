# GLDN Ops v3.4.8

Release date: 2026-07-05

## Summary

This release prepares GLDN Ops for multi-computer rollout with a local helper, popup-only access for less common workflows, cleaner daily panels, and 60-day duplicate prevention for bulk product sourcing.

## What Changed

- The floating daily panels are now simpler. Move .99, Bulk Listing Workflow, and Sniping Workflow are no longer shown there.
- Those workflows are still available from the extension popup.
- The popup now shows whether the local click helper is running.
- Bulk Listing Workflow keeps a 60-day per-computer product history to avoid sourcing the same Amazon item again too soon.
- EcomSniper Extract Sellers automation uses the local helper instead of Chrome debugger permission.
- eBay next-page handling uses the pagination URL when possible, reducing scroll-related failures.

## Operator Notes

- Start the local helper before running Bulk Listing Workflow:
  ```powershell
  powershell -NoProfile -ExecutionPolicy Bypass -File tools\local-click-helper.ps1
  ```
- In the extension popup, verify `Local click helper` says `Running`.
- Use `Start Bulk Listing Workflow` from the popup, not the floating page panel.
