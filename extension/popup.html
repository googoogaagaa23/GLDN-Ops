Feature guide

Short operator steps for the main GLDN Ops workflows.

Daily panel
  1. Use Side or minimize when the panel blocks the page.
  2. Use Reload Ext after updates.
  3. Use Stop Task to stop at the next safe checkpoint.
  4. Use Reset only when a saved workflow is stuck.
  5. Use Dashboard to open the shared dashboard.
Mark as shipped
  1. Open the eBay orders page.
  2. Click Mark as Shipped.
  3. Review any eBay confirmation before continuing.
Prepare order note
  1. Open Amazon checkout/order info and copy Amazon info.
  2. Open the matching eBay order.
  3. Click Prepare Order Note.
  4. Review the note, then Fill Add Note Box.
  5. Manually click eBay Save after reviewing.
Seller level and limits
  1. Open Seller Hub performance or overview.
  2. Run Scan Seller Level or Confirm Listings Under Limit.
  3. Review detected values before saving.
  4. Saved values sync to the dashboard and Tasks sheet.
  5. Metric alert rows show CHECK plus the affected computer names.
Move .99
  1. Set exact source and destination Store category names.
  2. Click Save .99 Categories for This eBay.
  3. Click Open Move .99 Workflow.
  4. Review the scan summary before applying changes.
  5. Stop at eBay review unless final Submit is approved.
Bulk listing and sniping
  1. Use Bulk Listing Workflow for broad Amazon Best Sellers sourcing.
  2. Use Sniping Workflow from a specific Amazon product with a visible price.
  3. Keep EcomSniper responsible for scanner, Product Hunter, and listing work.
  4. Keep the local helper running for automatic EcomSniper clicks.
Popup tools
  1. Open Amazon Best Sellers starts manual product research.
  2. Open EcomSniper Competitor Scanner opens EcomSniper research.
  3. Open EcomSniper Product Hunter opens the EcomSniper product tool.
  4. Check Local Helper confirms automatic EcomSniper clicks are available.
Settings
  1. Save Computer and eBay Account once per Chrome profile.
  2. Set Interface theme and transparency for GLDN Ops windows.
  3. Save Amazon Profile inside Amazon Chrome profiles only.
  4. Use Test Connection before trusting dashboard sync on a new install.
Diagnostics
  1. Open this popup after an error.
  2. Click Copy Error Log.
  3. Send the copied text with the page and button used.
