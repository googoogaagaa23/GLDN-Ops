GLDN Ops Feature Guide

Step-by-step instructions for every current feature.

Safety rule: GLDN Ops should stop before final eBay listing Submit actions unless the operator approves.

Setup

  1. Install GLDN Ops in the correct Chrome profile.
  2. Open the GLDN Ops popup.
  3. Select the computer number.
  4. Select the eBay account for that Chrome profile.
  5. Click Save Computer & eBay Account.
  6. Click Check Local Helper if using EcomSniper automation.

Floating Panel

  1. Use Side to move the panel out of the way.
  2. Use the minus button to minimize it.
  3. Use Dashboard to open the shared dashboard.
  4. Use Stop Task to stop a running workflow at the next safe checkpoint.
  5. Use Reset after a crash or stuck workflow.
  6. Use Reload Ext after updating the extension.

Mark as Shipped

  1. Open the eBay awaiting-shipment orders page.
  2. Click Mark as Shipped.
  3. Review any eBay confirmation dialog.
  4. Continue only if the selected orders are correct.
  5. GLDN Ops records the run locally and syncs it to the dashboard when available.

Prepare Order Note

  1. Open the Amazon checkout or order page.
  2. Click Review & Copy Amazon Info.
  3. Review total and ETA, then copy the info.
  4. Open the matching eBay order details page.
  5. Click Prepare Order Note.
  6. Review the editable note.
  7. Click Fill Add Note Box.
  8. GLDN Ops opens More actions, opens Add note, pastes the note, and waits for manual Save.

Scan Seller Level

  1. Open Seller Hub performance or click Scan Seller Level.
  2. Review the detected metrics.
  3. Click Save Seller Level Check.
  4. GLDN Ops syncs values to the dashboard and Tasks sheet.
  5. Tasks sheet alerts show CHECK computer when a metric crosses threshold.

Confirm Listings Under Limit

  1. Open Seller Hub Overview.
  2. Click Confirm Listings Under Limit.
  3. Review active listings, in-stock quantity, monthly dollar amount, and limits.
  4. Click Confirm Listings This Month.
  5. GLDN Ops records the monthly confirmation and syncs it.

Move .99 Listings

  1. Open the GLDN Ops popup.
  2. Set exact source Store category names.
  3. Set exact destination Store category name.
  4. Optional: add source category IDs.
  5. Click Save .99 Categories for This eBay.
  6. Click Open Move .99 Workflow.
  7. GLDN Ops opens Active Listings and scans matching listings.
  8. Review the scan summary before applying.
  9. GLDN Ops pauses at the eBay review screen before final Submit.

Bulk Listing Workflow

  1. Open the popup and click Start Bulk Listing Workflow.
  2. Use broad Amazon Best Sellers product/category research.
  3. Avoid clothing, shoes, and excluded categories.
  4. GLDN Ops assists the eBay search and EcomSniper Extract Sellers handoff.
  5. Continue scanner, Product Hunter, and listing work inside EcomSniper.

Sniping Workflow

  1. Open an Amazon product with visible price.
  2. Click Start Sniping Workflow.
  3. GLDN Ops searches eBay from that product context.
  4. Keep only likely dropshippers with enough markup.
  5. Use EcomSniper to scan competitors and identify winners.

Diagnostics

  1. Open the popup after an error.
  2. Click Copy Error Log.
  3. Send the copied log with the page and button used.
  4. Click Clear Error Log after the issue is captured.

Updating

  1. For unpacked installs, download/extract the newest GitHub ZIP.
  2. Open chrome://extensions.
  3. Click Reload on GLDN Ops, or load the new extension folder.
  4. Refresh the eBay/Amazon/EcomSniper work tab.
